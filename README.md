## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/laurent-kouassi/usersprofile.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. npm start
   ```sh
   npm start
   ```
